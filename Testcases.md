##Joining a game
###Test1: Single client joining:
Input:

```json
{"command":"join"}
```

Expected Response:
```json
{"pid":1,"type":"join"}
```

###Test2: Two clients joining:

```json
{"command":"join"}
{"command":"join"}

```

Expected Response:
```json
{"pid":1,"type":"join"}
```
```json
{"pid":2,"type":"join"}
```

###Test3: Three clients joining (More than the limit):
Input:
```json
{"command":"join"}
{"command":"join"}
{"command":"join"}
```

Expected Response:
```json
{"pid":1,"type":"join"}
```
```json
{"pid":2,"type":"join"}
```
```json
{"pid":-1,"type":"join"}
```
