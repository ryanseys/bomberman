##All integer representations on game are as follows:
0 = EMPTY
1 = PLAYER_1
2 = PLAYER_2
3 = PLAYER_3
4 = PLAYER_4
5 = ENEMY
6 = BOMB
7 = DOOR
8 = BOX
9 = POWERUP

##Joining a game
###Test1: Single client joining:
Input:

```json
{"command":"join"}
```

Expected Response:
```json
{"pid":1,"type":"join"}
```

###Test2: Two clients joining:

```json
{"command":"join"}
{"command":"join"}

```

Expected Response:
```json
{"pid":1,"type":"join"}
```
```json
{"pid":2,"type":"join"}
```

###Test3: Three clients joining (More than the limit):
Input:
```json
{"command":"join"}
{"command":"join"}
{"command":"join"}
```

Expected Response:
```json
{"pid":1,"type":"join"}
```
```json
{"pid":2,"type":"join"}
```
```json
{"pid":-1,"type":"join"}
```
